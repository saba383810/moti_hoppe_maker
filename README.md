# 🍡 もちほっぺ・メーカー

すきな2D画像のほっぺを、もちもち引っぱって遊べるWebおもちゃ。
「トリッカル・もちもちほっぺ大作戦」のほっぺ引っ張りの気持ちよさを目指しています。

**▶ https://saba383810.github.io/moti_hoppe_maker/**

## あそびかた

1. **いれる** — 画像をドロップ / ファイル選択 / ペースト（サンプルですぐ遊ぶこともできます）
2. **ぬる** — もちもちにしたいところ（ほっぺ）をブラシで塗る。「ぜんぶもちもち」でスキップも可
3. **あそぶ** — つまんで引っぱる。離すとぷるんと揺れて戻ります。マルチタッチで両ほっぺ同時もOK

⚙から、やわらかさ・揺れ具合などの調整、効果音の音量、録画（WebM動画 / GIF）ができます。

## 技術メモ

- Vite + TypeScript、フレームワークなし。レンダリングは WebGL2 生実装
- 画像をグリッドメッシュに貼り、変形は **制御点スプリング + 解析的変位場** 方式で頂点シェーダ計算
  - つまみは `tanh` で飽和（引くほど抵抗が増す）、falloff は quintic smootherstep
  - ドラッグ追従・揺れ戻りとも同一の減衰スプリング積分器。リリース時に速度が連続するため、フリックするとぷるんと飛ぶ
- ほっぺマスクはオフスクリーンcanvasにソフトブラシで描き、R8テクスチャとして頂点シェーダで変形weightに乗算
- 効果音はWebAudioでリアルタイム合成（音素材なし）
- WebGL2非対応ブラウザには案内カードを表示（WebGL1フォールバックはコストに見合わないため非対応）

## 開発

```sh
npm install
npm run dev      # 開発サーバ
npm run build    # 型チェック + 本番ビルド
npm run preview  # 本番ビルドの確認
```

`?debug` をURLに付けるとFPSとgrab数を表示します。

## デプロイ（GitHub Pages）

main へ push すると GitHub Actions が自動でビルド・デプロイします。
初回のみリポジトリ設定が必要です:

1. GitHub リポジトリの **Settings → Pages**
2. **Source** を **GitHub Actions** にする

`vite.config.ts` の `base: '/moti_hoppe_maker/'` はリポジトリ名と一致させてください。
